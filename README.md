# Hackathon_kstm
2023年9月に開催された部内ハッカソンで作成したWebアプリ

## LT会発表資料閲覧サービス

### 概要説明
- 毎月開催されるLT会において、発表者が作成した資料をkstmメンバーが閲覧できるWebサービス

### 使用環境
- とりあえずローカルで所望の動作を実現できるようにする
- 後々サーバにデプロイできるようにしたい
